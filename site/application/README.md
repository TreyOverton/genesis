## Phase I component overview

Every Phase I application consists of:

1. **SF-424 (R&R)** — Grants.gov cover form → [Forms & Administrative](forms.md)
2. **Project Summary/Abstract** — 1 page, public → [Project Summary](project-summary.md)
3. **Single PDF** attached to Field 8 of the R&R Other Project Information form, containing:
   - DOE Title Page (not counted toward page limit) → [DOE Title Page](doe-title-page.md)
   - Project Narrative (5 pages) → [Project Narrative](project-narrative.md)
   - Appendices 1–7 (not counted toward page limit) → [Appendix 1](appendix-1-bibliography.md) through [Appendix 7](appendix-7-letters.md)
4. **Senior/Key Person Profile (Expanded)** — biosketches + current & pending → [Personnel](personnel.md)
5. **R&R Budget + Justification** — per institution → [Budget](budget.md)
6. **R&R Subaward Budget Attachment** — if applicable
7. **Project/Performance Site Location(s)** — form
8. **Disclosure of Lobbying Activities (SF-LLL)** — form
9. **Identification of Merit Reviewer Conflicts** — submitted in PAMS after the application
10. **Genesis Mission Phase I Application Template (Excel)** — attached to Field 12; team + budget + computational resource summary
