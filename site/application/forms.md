# Forms & Administrative

> **Status:** Not started · **Owner:** Unassigned

## Required Grants.gov forms

| Form | Purpose | Key notes |
|------|---------|-----------|
| **SF-424 (R&R)** | Application cover | Type of Submission: **Application**. Type of Application: **New** |
| **R&R Budget** | Itemized budget per institution | See [Budget](budget.md) |
| **R&R Subaward Budget Attachment** | One per subrecipient | Only if applicable |
| **R&R Senior/Key Person Profile (Expanded)** | Key personnel + biosketches + C&P | See [Personnel](personnel.md) |
| **R&R Other Project Information** | Field 7 = Project Summary; Field 8 = single narrative PDF; Field 12 = Genesis Mission Excel template | See [Project Summary](project-summary.md), [Project Narrative](project-narrative.md), [DOE Title Page](doe-title-page.md) |
| **Project/Performance Site Location(s)** | Where work happens |  |
| **Disclosure of Lobbying Activities (SF-LLL)** | Federally required |  |

## Registrations (start early — multi-week lead time)

- **UEI (Unique Entity Identifier)** — applicant + each subrecipient
- **SAM.gov** — active registration, applicant + each subrecipient
- **Grants.gov** — workspace created
- **PAMS** (DOE Portfolio Analysis and Management System) — account verified

See [V.B in the RFA](../rfa/pages/5-submission-requirements-and-deadlines.md).

## PAMS submissions (post-application)

- **Identification of Merit Reviewer Conflicts** — list of individuals who should NOT serve as reviewers. Submitted in PAMS after the Grants.gov application. See [IX.B.10](../rfa/pages/9-other-information.md).

## Key dates

| Milestone | Date |
|-----------|------|
| Letter of Intent (LOI) | *Not required — Phase I only* |
| Pre-application | *Not required* |
| **Application submission** | **2026-04-28, 11:59 PM Eastern** |
| Late submissions | DOE reserves the right to decline without review |

See [V.D in the RFA](../rfa/pages/5-submission-requirements-and-deadlines.md).

## Submission checklist

- [ ] UEI active (applicant + all subs)
- [ ] SAM.gov current (applicant + all subs)
- [ ] Grants.gov workspace created
- [ ] PAMS account verified
- [ ] All required forms attached in Grants.gov
- [ ] Single narrative PDF assembled and validated (see [DOE Title Page](doe-title-page.md) for PDF rules)
- [ ] Genesis Mission Phase I Application Template (Excel) completed and attached to Field 12
- [ ] Project Summary attached to Field 7
- [ ] Budget Justification attached to Field L of R&R Budget
- [ ] Merit Reviewer Conflicts list queued in PAMS
